# 🐧 Linux Toolbox

A simple and useful Linux toolbox application made with Python and GTK 3.

Linux Toolbox provides quick access to helpful Linux commands and tools in an easy-to-use graphical interface.

## ✨ Features

* 🛠️ Useful Linux commands and tools
* 📁 Files & Folders tools
* 💾 Disk & Storage tools
* 👥 Users & Groups tools
* ⭐ Favorites
* 🖥️ Simple GTK graphical interface
* 🐧 Designed for Linux

## 📦 Installation

### Linux Mint / Ubuntu

Download the latest `.deb` package from the [Releases](../../releases) page.

Then install it by double-clicking the `.deb` file, or run:

```bash
sudo apt install ./linux-toolbox.deb
```

After installation, launch Linux Toolbox from your application menu or run:

```bash
linux-toolbox
```

## 🚀 Run from Source

Clone the repository:

```bash
git clone https://github.com/PROCHAMP-5110/Linux-Toolbox.git
cd Linux-Toolbox
```

Run the application:

```bash
python3 linux_toolbox.py
```

## 🛠️ Requirements

* Linux
* Python 3
* GTK 3
* PyGObject (`python3-gi`)
